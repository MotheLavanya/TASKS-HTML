What is Browser?

A Browser is a software applicaction that allows you to access and view web pages on the internet.It acts as an interface between you and the server.

Examples: Google Chorme,Mozilla,Firefox, and Apple Safari

What is Internet?

The internet is a global network that connects millions of computers and devices, allowing them to share information and communicate with each other.

Examples of Using the Internet: Sending an email, Watching videos, Searching for information, Using social media, Online shopping

What is URL?

A URL(Uniform Resource Locator) is the address used to access a specific resource on the internet, such as a website, image, or video.

Example:https://www.google.com

What is Domain?

Domain refers to the address or location of a website or a web resource.

Example:www.amazon.com

sub domain : www

main domain : amazon

Top-level domain: .com

What is Website and Web App?

Website: A website is a collection of web pages that you can visit through a browser.

Example: Google Chrome, Safari

Web App: A web app is a more interactive site where you can do things like fill out forms, make purchases, perform tasks. It's like a software application that runs in yours brower.

Gmail, Google Docs

What is HTTP?

HTTP stands for HyperText Transfer Protocol. It is the system used by web browsers and servers to communicate and share information over the internet.

Example: http://www.wikipedia.org

What is HTTPS?

HTTPS stands for HyperText Transfer Protocol Secure. It is a secure version of HTTP, used for safely transferring data between your browser and a website.

Example: https://www.amazon.com

What is IP address

An IP address(internet Protocol address) is a unique number assigned to every device connected to the internet.

Example: Google IP address: 142.250.190.78

What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create and design the structure of web pages.

Example: h1 shows a main heading of a page

What is XHTML?

XHTML stands for Extensible HyperText Markup Language. It is a stricter and cleaner version of HTML based on XML.

Example: All tags must be closed

What is Syntax?

Syntax refers to the rules and structure used to write the code properly.

Example: start tag--Content--end tag
